# Avaliação Técnica | L2Code

Repositório contendo a resolução da avaliação técnica proposta pela **L2Code**.

---

### Questão 1 - Loja do seu Manoel

Desenvolvimento de um algoritmo capaz de:

- Calcular a menor embalagem possível para uma compra;
- Considerar diferentes rotações dos produtos;
- Encontrar a melhor caixa entre opções pré-definidas;
- Funcionar com diferentes conjuntos de produtos e dimensões;
- Implementação em JavaScript com testes automatizados utilizando Vitest.

**Tecnologias**
- JavaScript 
- Node.js
- Vitest

---

### Questão 2 - Ajude o Professor

Implementação das consultas SQL solicitadas a partir do modelo entidade-relacionamento fornecido.

Consultas desenvolvidas:

- Total de horas comprometidas por professor;
- Lista de salas livres e ocupadas utilizando JOINs.

Conceitos utilizados:

- INNER JOIN
- LEFT JOIN
- GROUP BY
- SUM()
- CASE
- Relacionamentos entre tabelas

---

### Questão 3 - Nova L2Code

Customização do site da L2Code utilizando como base o projeto original.

A temática escolhida foi:

**Vendedor de Pipoca**

## Executando o projeto

Clone o repositório:

```bash
git clone https://github.com/JVsilveira/TecnicoL2.git
```

Entre na pasta:

```bash
cd LojaSeuManoel
```

Instale as dependências:

```bash
npm install
```

Execute os testes:

```bash
npm test
```

Execute o exercício:

```bash
node main.js
```
